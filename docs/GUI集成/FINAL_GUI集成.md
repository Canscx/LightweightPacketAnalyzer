## 架构设计

### 系统架构
```
GUI层 (MainWindow)
    ↓ 控制接口
业务逻辑层 (PacketCapture, DataProcessor, DataManager)
    ↓ 数据队列
数据传递层 (PacketQueue, StatsQueue)
```

### 数据流设计
1. **数据包捕获**: PacketCapture → PacketQueue → GUI显示
2. **统计处理**: DataProcessor → StatsQueue → GUI更新
3. **数据存储**: DataManager ← 数据包信息 ← DataProcessor

### 线程模型
- **主线程**: GUI界面和用户交互
- **捕获线程**: 数据包捕获和处理
- **定时器线程**: GUI定时更新
